App Function


Our app features all the user stories and needs. We designed a site that allows the user to login to their Spotify profile to see their Spotify Wrapped anytime. The users can see their top artists and songs and filter by time range and adjust the number of results per page.

Specifically, the web app features a home page that ranks the favorite songs and artists of users with a "Pop Score" of the artists to gauge their popularity. The favorite artists page lists the favorite artists of the user with their genre and Pop Score. The favorite tracks page lists the favorite songs of the user with the artist and album. The Wrapped portion gives the user a pop up tab that is in Spotify Wrapped format and allows the users to save and delete new and old wraps. The logout and delete account feature allows the user to logout and the delete account adds the functionality of also deleting the saved wraps. Furthermore, the web app also allows for light and dark mode, different languages, and a feature that allows the web app to play songs directly on Spotify. The web app directly aligns with the user story: "As a user, I want to be able to view a presentation of the different aspects of my personal Spotify music listening tastes displayed in colorful and fun ways."
